Introduction

The Sick Bees Model is an Individual Based Model, a type of Agent Based Model where agents have individual traits.

The computer simulation was written in Rust.

It models the spread of pathogens in wild bee populations.

Individual Based Modelling

Individual Based Models (IBM) are used to model systems with complex behavior.

The interaction of individuals leads to emergent patterns.

The Sick Bees Model

The Sick Bees model is written in Rust. There are several species classes that, e.g. hoverflies:

Unfolding

Calculate the path density of the unfolded accessibility graph

unfold

Unfolding uses the following algorithm:

  1. Add new nodes from contact graph to accessibility graph
  2. Add contact edges to accessibility graph
  3. Add path edges between nodes connected by a path of length t
  4. Count the number of path edges in the accessibility graph
  5. Normalize using the node count squared
